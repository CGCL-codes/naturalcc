# Code Generation Subtree

This directory mirrors a simplified version of the code-generation layout used in the project workspace.

Included here are:

- Lightweight evaluation helpers.
- Token-mapping utilities used for affix and vocabulary experiments.
- Minimal configuration files and example shell entry points.

Not included here are:

- Full model training pipelines.
- Internal transfer pipelines and unpublished algorithmic components.
- Large-scale evaluation scripts and result export workflows.
